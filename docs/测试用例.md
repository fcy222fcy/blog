## 一、认证模块测试用例

### TC-1.1 管理员登录 - 成功

| 项目 | 内容 |
|------|------|
| 用例编号 | TC-1.1 |
| 测试目标 | 验证管理员使用正确账号密码登录 |
| 优先级 | P0 |
| 前置条件 | 服务已启动，数据库已初始化 |

**测试步骤：**

1. 发送 POST 请求到 `/auth/login`
2. 请求体：
```json
{
  "username": "admin",
  "password": "123456"
}
```

**预期结果：**
- HTTP 状态码：200
- 响应体：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "expires_at": 1719000000
  }
}
```
- token 不为空
- expires_at 为未来时间戳
