📖 使用说明

设计特点

  • 品牌一致性：使用博客主题配色方案（#1B365D、#117361、#6d1b85）
  • 响应式设计：适配各种邮件客户端和移动设备
  • 视觉层次：清晰的信息层次和视觉引导
  • 行动号召：明确的 CTA 按钮，提高点击率

颜色方案

#1B365D
主色调 - 网站搭建
#117361
辅助色 - 生活记录
#6d1b85
强调色 - 软件开发

变量说明

变量 说明 示例
{{用户名}} 用户显示名称 张三
{{文章标题}} 文章完整标题 Golang 并发编程指南
{{文章摘要}} 文章简短摘要 深入浅出讲解 Golang 的并发模型...
{{文章链接}} 文章完整 URL https://blog.com/post/123
{{重置链接}} 密码重置 URL（含 token） https://blog.com/reset?token=xxx
{{评论内容}} 评论或回复的文本内容 这篇文章写得很好！
{{发布日期}} 文章发布日期 2026-07-04

技术要点

  • 兼容性：使用 Table 布局确保在各邮件客户端正常显示
  • 内联样式：所有 CSS 样式都使用内联方式，避免被过滤
  • 最大宽度：邮件内容宽度设置为 600px，适合大多数邮箱
  • 安全链接：所有外部链接都应该使用 HTTPS
  • 退订选项：每封邮件底部都包含取消订阅链接（法律要求）

测试建议

  • 在主流邮件客户端测试：Gmail、Outlook、Apple Mail、QQ邮箱等
  • 测试移动端显示效果
  • 检查所有链接是否可点击
  • 确认图片加载正常（如果使用了图片）
  • 测试暗黑模式兼容性

Go 后端集成示例

package email import ( "bytes" "html/template" "net/smtp" ) type WelcomeEmailData struct { Username string BlogURL string AboutURL string } func SendWelcomeEmail(to string, data WelcomeEmailData) error { // 读取模板 tmpl, err := template.ParseFiles("templates/welcome.html") if err != nil { return err } // 渲染模板 var body bytes.Buffer if err := tmpl.Execute(&body, data); err != nil { return err } // 发送邮件 auth := smtp.PlainAuth("", "your-email@example.com", "password", "smtp.example.com") msg := []byte("To: " + to + "\r\n" + "Subject: 欢迎加入 Fcy 的博客\r\n" + "MIME-Version: 1.0\r\n" + "Content-Type: text/html; charset=UTF-8\r\n" + "\r\n" + body.String()) return smtp.SendMail("smtp.example.com:587", auth, "your-email@example.com", []string{to}, msg) }

推荐邮件服务

  • SendGrid：功能强大，免费额度每天 100 封
  • Mailgun：开发者友好，前 5000 封免费
  • 阿里云邮件推送：国内访问快，价格便宜
  • 腾讯云 SES：每月 1000 封免费额度
  • Resend：现代化 API，免费额度每月 3000 封
💡 最佳实践提示：
  • 避免使用过多图片，可能被标记为垃圾邮件
  • 文案简洁明了，突出核心信息
  • CTA 按钮颜色要醒目，与背景对比度高
  • 保持邮件大小在 100KB 以内
  • 定期清理无效邮箱地址，维护发送信誉
  • 遵守反垃圾邮件法规，必须提供退订选项
