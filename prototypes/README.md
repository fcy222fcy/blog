# 原型图与设计文档

本文件夹用于存放项目的设计文档、原型图和实现计划。

## 目录结构

```text
prototypes/
├── README.md              # 本说明文件
├── frontend/              # 前台页面原型
│   ├── index.html         # 前台页面
│   ├── styles.css         # 样式文件
│   ├── script.js          # 脚本文件
│   └── avatar.jpg         # 头像图片
├── admin/                 # 后台管理原型
│   ├── index.html         # 后台页面
│   ├── css/
│   │   └── style.css      # 样式文件
│   └── js/
│       └── app.js         # 脚本文件
├── plans/                 # 实现计划文档
│   └── 2026-06-01-static-hugo-stack-clone.md
├── mockups/               # 原型图/UI设计稿 (待添加)
└── assets/                # 设计资源文件 (待添加)
```

## 文件命名规范

- 计划文档：`YYYY-MM-DD-描述.md`
- 原型图：`功能名称-版本.png/svg`
- 设计稿：`页面名称-状态.fig/sketch`
